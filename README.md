# mai-ead-enigne
enigne is the next powerful engine

## development

### requirements
* `cmake` version >= `3.21`
* `compiler` version  
* `gcc` / `g++` compiler

### spdlog library
install spdlog library for your platform: https://github.com/gabime/spdlog

### compile project

* create `./build` directory and `cd build`
* execute `cmake .. && make`


### run project
* `./Sandbox/Sandbox`
